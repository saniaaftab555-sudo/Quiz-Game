# Python Quiz Application

A simple command-line Quiz Application built using Python. This project allows users to answer multiple-choice questions and receive a score based on their performance.

## Overview
The Python Quiz Application is an interactive console-based program that presents users with multiple-choice questions. Users select answers from the given options, and the program validates each response before calculating the final score.

This project is beginner-friendly and helps in understanding core Python programming concepts such as loops, functions, conditionals, lists, dictionaries, and user input handling.

## Features
- Multiple-choice quiz questions
- Four answer options for each question
- User input validation
- Real-time answer checking
- Automatic score calculation
- Final score display

## Technologies Used
- Python 3

## How It Works
1. The program starts by loading quiz questions.
2. Each question is displayed with four answer choices.
3. The user enters an answer (A/B/C/D).
4. The program checks whether the answer is correct.
5. Score increases for each correct answer.
6. Final score is displayed after all questions are completed.

## Installation
1. Clone this repository:
```bash
git clone <repository-link>
```

2. Navigate to the project folder:
```bash
cd quiz_application
```

3. Run the program:
```bash
python quiz.py
```
